Redes y seguridad Informatica

Conectividad y redes IP

Este curso busca familiarizar al alumno en la implementacion de redes, proporcionandole la capacitacion necesaria para el cableado estructurado de los cables UTP, tambien en la instalacion y configuracion de servidores.

Windows server 2016

Este curso proporciona capacitacion y conocimientos para el uso de Windows server 2016 como la implementacion de una red de dominio o llevar a cabo las tareas administrativas en una red basada en dicho sistema.

Linux

El curso de linux consiste en aprender lo diferentes comandos de la consola con el fin dominar el sistema y realizar configuraciones avanzada, comprendiendo los conceptos fundamentales del sistema linux.

Programacion

Algoritmos y estructura de datos

El curso consiste en fundamentar los metodos de ciencia algoritmica y la parte aplicativa utilizando herramientas informaticas para visualizar una secuencia de elementos y condicionales.

Programacion en java

El curso de java tiene como proposito el desarrollo programas para computadoras mediante POO(Progamacion Orienta a Objetos) y estructura de datos. Los temas a tratar son: Clases, metodos, objetos, condicionales.

Programacion en PHP

El curso de lenguaje PHP busca familiarizar al alumno para el desarrollo web combinandolo con HTML.

Hardware

Ensamblaje de computadoras

Curso busca familiarizar al alumno estudiando las diferentes partes de una pc. Posibilitando al alumno realizar ensamblaje y poner en funcionamiento la computadora ensamblada

Reparacion de laptops y computadoras

Este curso consiste en realizar mantenimiento fisico o lo logico en una laptop o computadora.

Instalacion y configuracion del SO

Este curso tiene como fin de generar la capacidad de instalar y configurar sistemas operativos de una computadora y solucionando conflictos de software asi como la creacion de copias de seguridad y instalacion de antivirus.
